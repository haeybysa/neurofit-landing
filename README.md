# NeuroFit — AI Fitness Coach & Sports Partner Finder

**NeuroFit** — умный фитнес-помощник с AI-коучем, поиском партнёров для ракеточных и командных видов спорта, а также бронированием спортивных площадок.

**NeuroFit** — 智能健身助手，配备AI教练、球拍类和团队运动伙伴搜索以及运动场地预订功能。

---

## 🚀 О проекте / 关于项目

**Русский:**  
NeuroFit объединяет персональный AI-коучинг, AR-анализ техники, адаптивное питание, поиск партнёров по видам спорта (теннис, падел, гольф, настольный теннис и др.) и бронирование кортов — всё в одном приложении.

**中文：**  
NeuroFit将个人AI教练、AR技术分析、自适应营养、按运动类型寻找伙伴（网球、板式网球、高尔夫、乒乓球等）以及场地预订整合到一个应用程序中。

## 🎯 Ключевые функции / 主要功能

| Функция / 功能 | Описание / 描述 |
|----------------|-----------------|
| 🤖 AI-коуч / AI教练 | Адаптирует тренировки под HRV, сон, нагрузку / 根据HRV、睡眠、负荷调整训练 |
| 📹 AR-анализ / AR分析 | Коррекция техники в реальном времени / 实时技术纠正 |
| 🍽️ AI-питание / AI营养 | Разбор макро по фото еды / 通过食物照片分析宏观营养 |
| 🤝 Поиск партнёров / 伙伴搜索 | По уровню, стилю, расписанию / 按水平、风格、时间表 |
| 🏟️ Бронирование / 预订 | Корты, поля, залы с QR-кодом / 带QR码的球场、场地、场馆 |
| 🏆 Геймификация / 游戏化 | RPG-прогрессия, кланы, PvP / RPG进阶、战队、PvP |
